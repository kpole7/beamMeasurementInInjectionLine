Tests can be performed in two modes:
- hardware mode (using specialized printed circuit boards);
- simulation mode (using only a microcontroller).
Both modes provide two USART interfaces for connecting to a computer.

1. Multi-channel input currents measurement (only hardware mode).
1.1. Make sure the software is running in the correct mode.
1.2. Send Modbus command "write single register"; address 1132; data 1 (analog inputs printouts).
1.3. Connect the reference current source to the current inputs and check the displayed values.
1.4. Change the active cup (send Modbus command "write single register"; address 1133; data 1...3); go to 1.3.

2. Reading logical input signals.
2.1. Hardware mode.
2.1.1. Make sure the software is running in the correct mode.
2.1.2. Send Modbus command "write single register"; address 1132; data 4 (logic inputs printouts).
2.1.3. Check the “sw1”, “sw2”, “sw3a”, and ‘sw3b’ inputs by pulling them low; 
       check the “INH” input by pulling it high (+5V).
